Cara Install Ubuntu 16.04

1. Klik Install Ubuntu




2. Jika anda terhubung ke internet, dan ingin meng-install driver
seperti graphic, Wifi, MP3 dan lainnya silakan centang pada pilihan kedua,
jida tidak langsung klik “Continue”.




3. Centang “Something else” untuk menentukan partisi dan alokasi sendiri




4. Pilih zona waktu yang anda inginkan dengan klik pada peta, lalu klik “Continue”.




5. Pilih layout keyboard yang sesuai dengan keyboard anda, lalu klik “Continue”.




6. Isikan detail user yang akan digunakan pertama kali, lalu klik “Continue”




7. Silakan tunggu sampai proses instalasi selesai.




8. Setelah proses instalasi selesai, anda dapat langsung me-restart PC
